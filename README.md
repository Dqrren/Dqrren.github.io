```
portfolio/
├── index.html       # Main page
├── style.css        # All styles

├── img/
│   └── profile.jpg  # Your profile photo (add this!)
└── README.md
```

## 🛠️ How to Deploy on GitHub Pages

1. Create a new repository named **`yourusername.github.io`**
   - e.g. `darrends.github.io`
2. Upload all files to the root of that repo
3. Go to **Settings → Pages → Source: main branch / root**
4. Your site will be live at `https://yourusername.github.io`

## 📸 Adding Your Profile Photo
- Add a photo named `profile.jpg` inside the `img/` folder
- Recommended: square crop, at least 600×600px
- Without the photo, the site shows a styled placeholder

## 📊 IDX Project — Publishing Guide
To publish the IDX dashboard project safely:
1. Add a `.gitignore` to exclude data files: `*.csv`, `*.xlsx`, `*.pbix`
2. Include screenshots of the dashboard in an `img/` or `assets/` folder
3. Write a thorough `README.md` describing your methodology
4. Recruiters care about your thinking, not the raw data
